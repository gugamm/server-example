# server-example
A setup example with:
* node
* mocha/chai
* pug

## Getting started

```bash
git clone https://github.com/gugamm/server-example
cd server-example
npm install
``` 

## Running server in watch mode

```bash
npm run swatch
```

Now go to http://localhost:3000

## Testing

```bash
npm run test
```

## Testing in watch mode

```bash
npm run testw
```
